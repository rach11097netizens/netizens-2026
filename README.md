# Netizens Technologies - Homepage

A high-performance, modern landing page built with React, TypeScript, Tailwind CSS, and GSAP animations.

## Features

- **Modern Tech Stack**: React 18 + TypeScript + Vite
- **Beautiful UI**: Tailwind CSS with custom design tokens
- **Smooth Animations**: GSAP ScrollTrigger for engaging scroll animations
- **Fully Responsive**: Optimized for all device sizes
- **Performance First**: Lightweight and fast-loading

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Technologies Used

- **React 18**: UI library
- **TypeScript**: Type safety
- **Vite**: Build tool and dev server
- **Tailwind CSS**: Utility-first CSS framework
- **GSAP**: Animation library
- **Lucide React**: Icon library

## Customization

- Update colors in `tailwind.config.js`
- Modify content in component files
- Add/remove sections in `App.tsx`
- Replace placeholder images with actual assets
